🏥 CityMed Hospital – ServiceNow Case Study (Days 1–20)
📌 Overview

This repository contains my implementation of the CityMed Hospital 20-Day ServiceNow Case Study using the ServiceNow Australia Release. The project demonstrates the development of a scoped application covering IT service management, equipment fault tracking, automation, security, reporting, and self-service capabilities using real-world ServiceNow concepts.

🚀 Features Implemented
Custom Scoped Application
Equipment Fault Management
Asset Import using Import Sets & Transform Maps
Users, Groups & Role Management
Access Control Lists (ACLs)
Update Sets & Release Management
Knowledge Base & System Properties
Reports & Dashboards
Service Catalog & Record Producers
Client Scripts & UI Policies
Script Includes & GlideAjax
Business Rules (Before, After & Async)
Flow Designer Automation
Email Notifications
Scheduled Jobs
End-to-End Workflow Testing
